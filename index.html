📋 Common Commands

Category Command Description
Install sudo apt install nginx -y Install Nginx package.
Validate sudo nginx -t Check configuration syntax.
Reload sudo systemctl reload nginx Apply changes without downtime.

⭐ Important Notes

★ Production Best Practices

  • Always test with nginx -t.
  • Use HTTPS for public websites.
  • Keep separate logs per site.
  • Use reload instead of restart when possible.
